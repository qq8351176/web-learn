### 流式布局(百分比布局)

百分比布局 通过设置盒子的宽度为百分比 根据屏幕的宽度来进行伸缩 不受固定像素的限制  是移动端web的常见布局

    *
    {
        margin:0;
        padding:0;
    }
    section
	{
		width:100%;
	}
    section div
    {
        float:left;
        width:50%;
        height:400PX;
        background-color:pink;
    }
</style>

    <div></div>
    <div></div>
</section>
